WHAT IS JAVASCRIPT



Javascript is a programming language that allows you to implement complex items in web pages - every time a page shows you something beyond a static information, showing content that is refreshed constantly, interactive maps or 2D/3D animated graphics, etc - there certainly Javascript is involved. It is the third layer of the web technologies followed by HTML and CSS.


WHAT DOES IT DO?

    Some of the many benefits of using Javascript:
  • Store useful content in variables: you can request the user for input (such as name, email, etc) and store it in its respective variable.
  • Operations using text or “strings” like in programming: an exemple is when you type your name in a website, it shows you a message adding your name to it (“Hi, John! How can I help you?”)
  • Execute codes with basis on actions such as a click, or keyboard input: when clicking on a link that says “Show More” and suddenly the text opens up in the same page instantly.
  • And much… much more!

THE APIs

Another great feature from Javascript are the APIs (Application Programming Interfaces) that provides you superpowers to implement and improve your code. They are codes ready to use to construct parts of a page allowing the developer to implement them in a much easier manner. They could be compared as furniture kits where you buy them disassembled and all that you have to do is assemble them rather than having to make all the measurements and buy the materials etc.
